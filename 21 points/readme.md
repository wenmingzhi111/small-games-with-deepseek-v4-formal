prompt is :
制作一款基于 21 点（Blackjack）的工业/奇幻恐怖风格 Roguelite 游戏，灵感源自《小丑牌》（Balatro）、《Cloverpit》、《邪恶冥刻》（Inscryption）以及《Slots and Daggers》。

核心要求如下：

文件结构：必须包含在单个 HTML 文件中。

视觉效果：采用 SVG 图形，并配有流畅的动画以增添视觉效果。

音效与音乐：必须包含音效和背景音乐。

进度系统：包含“单局特定解锁（局内升级）”和“永久解锁（局外养成）”，以及成就系统。

关卡设计：具有类似《杀戮尖塔》（Slay the Spire）的关卡/路线选择器。

随机与种子：包含大量基于种子的 RNG（随机数生成）元素，并支持输入种子进行游戏。

升级与构筑：提供强调协同效应（Synergy）和支持多种玩法的升级项目。

耐玩度：具备极高的可重玩价值。

难度与模式：提供不同的难度预设，以及可解锁的初始天赋/模式。

how to play?
《血工坊 · BLOODWORKS 21》 工业/奇幻恐怖 21点 Roguelite，已实现全部要求：
- 单文件：所有 HTML/CSS/JS 内联，无外部依赖。
- SVG + 动画：SVG 绘制卡牌/地图/图标；发牌翻转、伤害弹字、节点脉冲、扫描线+噪点滤镜。
- 音效与音乐：Web Audio 程序化生成——工业鼓点+drone+小调琶音循环 BGM，及发牌/要牌/爆牌/暴击/金币/解锁等音效（可静音）。
- 进度系统：局内血契升级（index.html:RELICS）+ 局外血晶养成（PERKS 7 项）+ 9 项成就（ACHS），localStorage 存档。
- 关卡选择器：《杀戮尖塔》式分列路线图，对决/精英/商店/营火/宝藏/首领分支，SVG 贝塞尔连线（buildMap/renderMap）。
- 种子 RNG：xmur3+mulberry32，牌序/地图/掉落全部种子化，支持输入种子；已验证同种子完全一致、异种子不同。
- 协同构筑：21 种血契围绕花色流/人头流/A流/低血狂血流/对子流等，筹码×倍率计分（《小丑牌》风）。
- 难度与模式：4 档难度（学徒→献祭）、5 个可解锁开局天赋、4 个可解锁模式（死斗/疾行/混沌）。
操作：HIT/STAND 按钮或键盘 H/S（空格要牌、回车停牌）。首次点击任意处启动音频。
